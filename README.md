## 🧩 建议业务流程

1. 初步咨询  
   用户描述问题类型，但不提供助记词、私钥、验证码。

2. 风险隔离  
   停止在可疑设备继续操作，保护仍可控制资产。

3. 资料收集  
   仅收集公开地址、交易哈希、钱包类型、设备状态、官方邮件截图等非敏感信息。

4. 技术分析  
   基于钱包类型、链上记录、设备状态、备份线索和算法模型进行恢复路径判断。

5. 算力验证  
   在用户授权范围内，使用 GPU 算力和多算法引擎进行候选验证与地址匹配。

6. 输出建议  
   提供恢复路径、材料清单、风险提示和下一步操作方案。
